## Overview

monaco-vim exposes the underlying CodeMirror vim implementation, allowing you to create custom keybindings and remap existing ones. This is useful for adding editor-specific functionality or customizing the vim behavior to match your workflow.

## Accessing the Vim API

The vim implementation is available through the `VimMode` export:

```javascript
import { VimMode } from "monaco-vim";

// Access the Vim API
VimMode.Vim.mapCommand(/* ... */);
VimMode.Vim.defineEx(/* ... */);
```

<Info>
  `VimMode.Vim` is the CodeMirror Vim implementation adapted for Monaco. It provides the same API as `CodeMirror.Vim` from the original CodeMirror vim mode.
</Info>

## Using mapCommand

The `mapCommand` function allows you to create custom key mappings that execute actions, motions, or operators.

### Function Signature

```typescript src/cm/keymap_vim.ts:6810-6816
function mapCommand(keys, type, name, args, extra) {
  var command = { keys: keys, type: type };
  command[type] = name;
  command[type + "Args"] = args;
  for (var key in extra) command[key] = extra[key];
  _mapCommand(command);
}
```

### Parameters

<ParamField path="keys" type="string" required>
  The key sequence to bind (e.g., "jj", `<C-s>`, `<Leader>w`)
</ParamField>

<ParamField path="type" type="string" required>
  The command type: "action", "motion", "operator", or "keyToKey"
</ParamField>

<ParamField path="name" type="string" required>
  The name of the action/motion/operator to execute
</ParamField>

<ParamField path="args" type="object">
  Arguments to pass to the command
</ParamField>

<ParamField path="extra" type="object">
  Additional properties like context ("normal", "insert", "visual")
</ParamField>

## Mapping Examples

### Key-to-Key Mappings

Remap one key sequence to another:

```javascript
import { VimMode } from "monaco-vim";

// Map 'jj' to Escape in insert mode
VimMode.Vim.mapCommand(
  "jj",
  "keyToKey",
  "<Esc>",
  {},
  { context: "insert" }
);

// Map 'H' to beginning of line
VimMode.Vim.mapCommand(
  "H",
  "keyToKey",
  "^",
  {},
  { context: "normal" }
);

// Map 'L' to end of line
VimMode.Vim.mapCommand(
  "L",
  "keyToKey",
  "$",
  {},
  { context: "normal" }
);
```

### Action Mappings

Map keys to vim actions:

```javascript
// Map Ctrl+s to save (requires defining a save action first)
VimMode.Vim.defineAction("save", function(cm, actionArgs) {
  // Your save logic here
  console.log("Saving file...");
});

VimMode.Vim.mapCommand(
  "<C-s>",
  "action",
  "save",
  {},
  { context: "normal" }
);
```

### Motion Mappings

Create custom motions:

```javascript
// Define a custom motion
VimMode.Vim.defineMotion("moveToNextFunction", function(cm, motionArgs) {
  // Custom motion logic
  var cursor = cm.getCursor();
  // ... find next function and return position
});

// Map it to a key
VimMode.Vim.mapCommand(
  "]f",
  "motion",
  "moveToNextFunction",
  { forward: true },
  { context: "normal" }
);
```

## Special Key Notation

<AccordionGroup>
  <Accordion title="Modifier Keys">
    - `<C-x>` - Control + x
    - `<A-x>` or `<M-x>` - Alt/Meta + x
    - `<S-x>` - Shift + x
    - `<D-x>` - Command (Mac) / Windows key
  </Accordion>
  
  <Accordion title="Special Keys">

    - `<CR>` - Enter/Return
    - `<BS>` - Backspace
    - `<Del>` - Delete

    - `<Space>` - Space

## Practical Examples

### Monaco-Specific Bindings

```javascript
import { VimMode } from "monaco-vim";

// Quick save with Ctrl+S
VimMode.Vim.defineAction("saveFile", function(cm) {
  const value = cm.editor.getValue();
  localStorage.setItem("editorContent", value);
  cm.openNotification("File saved!");
});

VimMode.Vim.mapCommand("<C-s>", "action", "saveFile");

// Format document with <Leader>f
VimMode.Vim.defineAction("formatDocument", function(cm) {
  cm.triggerEditorAction("editor.action.formatDocument");
});

VimMode.Vim.mapCommand("<Leader>f", "action", "formatDocument");

// Quick comment toggle with gc
VimMode.Vim.defineAction("toggleComment", function(cm) {
  cm.triggerEditorAction("editor.action.commentLine");
});

VimMode.Vim.mapCommand("gc", "action", "toggleComment");
```

### Insert Mode Convenience

```javascript
// Emacs-style cursor movement in insert mode
VimMode.Vim.mapCommand("<C-a>", "action", "goLineStart", {}, { context: "insert" });
VimMode.Vim.mapCommand("<C-e>", "action", "goLineEnd", {}, { context: "insert" });
VimMode.Vim.mapCommand("<C-f>", "keyToKey", "<Right>", {}, { context: "insert" });
VimMode.Vim.mapCommand("<C-b>", "keyToKey", "<Left>", {}, { context: "insert" });
```

<Warning>
  Custom mappings are global and will affect all editor instances using monaco-vim. Define them once during application initialization.
</Warning>

## Removing Mappings

To remove or override a default mapping, simply define a new mapping with the same key sequence. The new mapping will take precedence.

```javascript
// Override the default 's' behavior
VimMode.Vim.mapCommand(
  "s",
  "action",
  "myCustomAction",
  {},
  { context: "normal" }
);
```
