# Quickstart Guide

This guide will walk you through setting up monaco-vim in your Monaco Editor instance. You'll have Vim keybindings working in just a few minutes.

## Complete Example

Here's a full working example of monaco-vim in action:

```javascript
import * as monaco from "monaco-editor";
import { initVimMode } from "monaco-vim";

// Create the editor
const editor = monaco.editor.create(document.getElementById("editor"), {
  value: `function hello() {
  console.log("Hello from Vim mode!");
}`,
  language: "javascript",
  theme: "vs-dark",
  minimap: { enabled: false },
});

// Initialize Vim mode with status bar
const statusbarNode = document.getElementById("statusbar");
const vimMode = initVimMode(editor, statusbarNode);

// Later, to remove Vim mode:
// vimMode.dispose();
```

## Step-by-Step Setup

<Steps>
  <Step title="Create HTML structure">
    Set up your HTML with containers for both the editor and the status bar:

    ```html
    <!DOCTYPE html>
    <html>
      <head>
        <meta charset="utf-8" />
        <title>Monaco Vim Example</title>
        <style>
          #editor {
            width: 800px;
            height: 600px;
            border: 1px solid #ccc;
          }
          #statusbar {
            padding: 5px;
            font-family: monospace;
            background: #f0f0f0;
            border: 1px solid #ccc;
            border-top: none;
          }
        </style>
      </head>
      <body>
        <div id="editor"></div>
        <div id="statusbar"></div>
        <script src="./src/main.js"></script>
      </body>
    </html>
    ```

    <Info>
    The status bar element displays the current Vim mode (NORMAL, INSERT, VISUAL) and key buffer.
    </Info>
  </Step>

  <Step title="Create and configure Monaco Editor">
    Initialize your Monaco Editor instance:

    ```javascript
    import * as monaco from "monaco-editor";

    const editor = monaco.editor.create(document.getElementById("editor"), {
      value: `var StringStream = function (string, tabSize) {
  this.pos = this.start = 0;
  this.string = string;
  this.tabSize = tabSize || 8;
};`,
      language: "javascript",
      theme: "vs",
      fontSize: 15,
      minimap: {
        enabled: false,
      },
      scrollBeyondLastLine: false,
    });
    ```

    <Tip>
    Configure Monaco Editor with your preferred settings before initializing Vim mode.
    </Tip>
  </Step>

  <Step title="Initialize Vim mode">
    Import and initialize monaco-vim with your editor and status bar:

    ```javascript
    import { initVimMode } from "monaco-vim";

    const statusbarNode = document.getElementById("statusbar");
    const vimMode = initVimMode(editor, statusbarNode);
    ```

    The `initVimMode` function returns a `VimAdapterInstance` that you can use to control the Vim mode.

    <Note>
    The status bar parameter is optional. If you pass `null`, Vim mode will work but without visual status feedback:
    
    ```javascript
    const vimMode = initVimMode(editor, null);
    ```
    </Note>
  </Step>

  <Step title="Start using Vim commands">
    Your editor now responds to Vim keybindings! Try these commands:

    - Press `i` to enter INSERT mode
    - Press `Esc` to return to NORMAL mode
    - Use `h`, `j`, `k`, `l` for navigation
    - Press `v` for VISUAL mode
    - Type `:` to enter command mode
    - Use `dd` to delete a line
    - Use `yy` to yank (copy) a line

    The status bar will show your current mode and key buffer.
  </Step>

  <Step title="Clean up when done (optional)">
    If you need to disable Vim mode later:

    ```javascript
    vimMode.dispose();
    ```

    This removes all Vim keybindings and restores default Monaco Editor behavior.
  </Step>
</Steps>

## TypeScript Example

If you're using TypeScript, you can type your code for better IDE support:

```typescript
import type * as MonacoEditor from "monaco-editor";
import * as monaco from "monaco-editor";
import { initVimMode, VimAdapterInstance } from "monaco-vim";

const editorNode = document.getElementById("editor") as HTMLElement;
const statusNode = document.getElementById("statusbar") as HTMLElement;

const editor: MonacoEditor.editor.IStandaloneCodeEditor = monaco.editor.create(
  editorNode,
  {
    value: "console.log('Hello, Vim!');",
    language: "javascript",
    theme: "vs-dark",
  }
);

const vimMode: VimAdapterInstance = initVimMode(editor, statusNode);
```

## Handling Key Events

If you need certain key combinations to bypass Vim mode (e.g., Cmd+Enter to run code), add your key handler before initializing Vim:

```javascript
editor.onKeyDown((e) => {
  // Prevent Vim from handling Cmd+Enter
  if ((e.metaKey || e.ctrlKey) && e.keyCode === monaco.KeyCode.Enter) {
    e.preventDefault();
    // Your custom logic here
    runCode();
  }
});

// Initialize Vim mode after setting up key handlers
const vimMode = initVimMode(editor, statusbarNode);
```

<Info>
monaco-vim ignores key events where `preventDefault()` has been called, allowing you to handle specific shortcuts yourself.
</Info>

## Multiple Editors

You can enable Vim mode on multiple editor instances independently:

```javascript
const editor1 = monaco.editor.create(document.getElementById("editor1"), {...});
const editor2 = monaco.editor.create(document.getElementById("editor2"), {...});

const vimMode1 = initVimMode(editor1, document.getElementById("status1"));
const vimMode2 = initVimMode(editor2, document.getElementById("status2"));
```

Each editor maintains its own independent Vim state.
